# Multi-Process-File-Synchronization-System-using-Pipes
• Built a multi-process system in C++ using fork() and pipes for inter-process communication  
• Implemented bidirectional message passing between processes using UNIX pipes  
• Designed a file synchronization mechanism between two directories  
• Handled partial failures by transmitting error states along with intermediate data  
• Implemented custom serialization/deserialization for structured data transfer
